# 🚀 TravelSmart

TravelSmart is a smart multimodal transport web application that helps users find the most efficient routes across a city using different transportation modes like bus, train, cab, auto, and walking.

---

## 🌍 Problem

In rapidly growing urban cities, passengers, students, and newcomers struggle with:
- ❌ Lack of integrated transport options
- ❌ Difficulty finding optimal routes
- ❌ Confusing navigation systems
- ❌ Inefficient travel planning
- ❌ Excessive walking distances

---

## 💡 Solution

TravelSmart simplifies urban travel by:
- ✅ Providing fastest and cheapest routes
- ✅ Supporting multiple transport modes
- ✅ Offering step-by-step route guidance
- ✅ Showing estimated time, cost, and distance
- ✅ Visualizing routes on an interactive map

---

## ✨ Features

- 🧭 Smart route suggestions (Fastest & Cheapest)
- 🗺️ Interactive map using Leaflet.js + OpenStreetMap
- 📍 Location search with geocoding (Nominatim API)
- 📊 Dynamic route generation (time, cost, distance)
- 🚶 Multimodal transport simulation:
  - Bus 🚌
  - Train 🚆
  - Cab 🚕
  - Auto 🛺
  - Walking 🚶
- ⏳ Loading state for better UX
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Leaflet.js (Map)
- OpenStreetMap API (Nominatim)
